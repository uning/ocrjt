About 微信聊天记录处理工具

微信聊天记录处理工具是一个用于将微信聊天记录中的图片保存到电脑，并通过文字识别技术整理数据的工具。以下是工具的一些功能和使用说明。

保存微信聊天记录图片到电脑

要保存微信聊天记录图片到电脑，请按以下步骤操作：

  1. 找到电脑版微信设置，迁移与备份，打开。
  2. 将手机和电脑连接到同一 Wi-Fi 网络，然后在手机微信上选择要同步的聊天记录。
  3. 选择聊天记录管理，点击保存到电脑。

运行

要运行工具，请按以下步骤操作：

  1. 复制 apiConfig.example.js 为 apiConfig.js。
  2. 配置 API 参数在 apiConfig.js。
  3. 配置需精准识别品名的商品名在 apiConfig.js。
  4. 运行命令：node your_script.js，其中 your_script.js 是你的脚本文件。
